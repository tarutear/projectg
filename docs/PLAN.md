# MVP 개발 플랜

자세한 내용은 각 Phase 구현 커밋 참조.

## 단계 요약

| 단계 | 내용 | 상태 |
|------|------|------|
| Phase 1-1 | 프로젝트 셋업 + 카메라 스트림 | ✅ 완료 |
| Phase 1-2 | OpenCV.js WASM + Web Worker | 예정 |
| Phase 1-3 | HSV 마커 검출 + 오버레이 | 예정 |
| Phase 1-4 | 마커 추적 고도화 + 라벨 UX | 예정 |
| Phase 1-5 | 각도/거리 계산 + 각도 그룹 UX | 예정 |
| Phase 1-6 | 세션 관리 + IndexedDB 저장 | 예정 |
| Phase 1-7 | 결과 시각화 + 통합 테스트 | 예정 |

## 의존성

```
1-1 → 1-2 → 1-3 → 1-4 → 1-5 ──병렬──→ 1-6
                               └────────── 1-7
```

## 성능 목표

- 20 FPS (권장 환경 기준)
- 처리 해상도: 640×480 다운스케일, 오버레이는 원본 해상도 역변환
- 세션 버퍼: 최대 10,000 프레임 circular buffer
