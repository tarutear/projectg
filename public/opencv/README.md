# OpenCV.js WASM

Place `opencv.js` (OpenCV.js 4.x WebAssembly build) in this directory.

The file is excluded from git due to its size (~8MB).
Download from: https://docs.opencv.org/4.x/d5/d10/tutorial_js_root.html

For a smaller build (~3MB), compile from source with only `core` and `imgproc` modules.
